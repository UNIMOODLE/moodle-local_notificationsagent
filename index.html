Agente de notificaciones

Acciones automáticas en Moodle

UNIMOODLE es un proyecto colaborativo creado por las universidades españolas de Valladolid, Complutense de Madrid, País Vasco/EHU, León, Salamanca, Illes Balears, Valencia, Rey Juan Carlos, La Laguna, Zaragoza, Málaga, Córdoba, Extremadura, Vigo, Las Palmas y Burgos  para desarrollar nuevas herramientas para mejorar los campus virtuales basados en Moodle.

¿Qué es Agente de notificaciones?


El plugin de Agente de Notificaciones para Moodle, el sistema de gestión del aprendizaje (LMS) de código abierto más popular del mundo, permite a los profesores configurar en sus asignaturas una serie de acciones automáticas para enviar mensajes de seguimiento a sus estudiantes.
Gracias a esto, los estudiantes pueden activar un conjunto de reglas para ser avisados específicamente cuando ocurran eventos específicos.
Tanto Administradores como profesores podrán configurar una serie de reglas para que los alumnos sean avisados cuando se den ciertas condiciones, excepciones y/o acciones. 

¿Por qué utilizarlo?


A lo largo de los cursos se dan una serie de eventos de los que es imprescindible que los alumnos estén informados. Aunque Moodle viene por defecto con una serie de reglas de notificación que se pueden activar, estas suelen ser insuficientes.
En el mercado, existen algunos otros plugins que permiten generar notificaciones pero o no están correctamente integrados con Moodle, conllevan una carga muy grande de procesos o no cubren todas las necesidades.
Con el plugin de Agente de notificaciones tanto los administradores como los profesores podrán realizar reglas de notificaciones para eventos de una forma ágil y completamente integrada con Moodle.
Las acciones que envían mensajes a los alumnos se pueden personalizar mediante comodines (nombre, apellidos…) dentro del texto a enviar. Los textos pueden tener redacciones distintas que se usarán en función de las veces que se haya ejecutado la regla.

Además, Agente de Notificaciones dispone de un informe para ver un histórico de cada activación de una regla.
Código abierto y gratuito: Agente de Notificaciones es un plugin para Moodle, que es un LMS de código abierto y gratuito. Esto significa que puedes utilizar Agente de Notificaciones sin ningún coste o cuota de licencia, y también puedes modificarlo, mejorarlo y compartirlo con otros, gracias al Consorcio de Universidades UNIMOODLE, que creó y mantiene este plugin.  

¿A quién va dirigido?


Agente de Notificaciones está enfocado a todo tipo de instituciones académicas y empresas que ofrezcan formación, ya que permite a los administradores y docentes poder informar a todos sus estudiantes de los próximos eventos de forma ágil y eficaz.

Agente de Notificaciones es compatible con Moodle 4.1 y superiores, y puede integrarse fácilmente con otras funciones y plugins de Moodle. 

Características

Diseñado para Moodle

Agente de Notificaciones ha sido desarrollado e integrado como plugin para Moodle, la mayor plataforma de e-learning del mundo.

Se integra con otras actividades (a través de la API de finalización) y con el libro de calificaciones.
Agente de Notificaciones está diseñado para usarse con dispositivos móviles.

Integrado en su sitio Moodle

Agente de Notificaciones se integra en la plataforma que mejor conocen tus profesores y alumnos.






Arquitectura de subplugins

Para poder extender la funcionalidad. Tanto para las condiciones como para las acciones es posible el desarrollo de subplugins para extender la funcionalidad del plug-in.

Notificaciones a la carta

El agente de notificaciones permite a los profesores crear reglas complejas a partir de una interfaz sencilla o a partir de plantillas y que ejecutarán una o varias acciones en el momento en que se cumplan las condiciones configuradas.
